Claude After Dinner

A collection of explorations using Claude, PDF and Web Audio API to build some tools that speed up everyday tasks.

Enhance Speech Mixer

Generate a video and audio file that mixes five seconds of alternating segments of the original and enhanced audio, making it easy to listen and evaluate model quality.

Waveform Generator

Upload a video file and add a waveform on top matching the audio playback. Customize the waveform color, width, and thickness before downloading it to share.

Export Analyzer

Compare file metadata of the enhanced files for V1, V2 across free and premium with the original track to identify properties that have changed.

Service Cost Estimator

Using average and peak requests calculate projected service cost based on requests, file duration and generated audio per minute for each month.

Transcript Comparison

Compare two transcripts side by side with highlights to indicate added and deleted words along with a summary of changes including a similarity score.

Transcript Analysis

Upload a transcript and easily highlight filler words, repeated words and numeric content. Ideal to review test files that could be used for specific scenarios.

Podcast Creator

Generate a Podcast by uploading a recording with music that is automatically added as a intro, outro and transition with fade in and outs. Transitions are added during silence or pause.

Realtime Audio Transcription

Upload an audio file and generate a transcription in realtime during playback.

Podcast Insights

Generate a summary, speaker highlights, chapters, conversation flows, category tags and several other insights from your Podcast recording in seconds.

Project Ten

Machine learning pipeline for image recognition and classification. Includes model training interface, API endpoints, and integration with popular deep learning frameworks.
